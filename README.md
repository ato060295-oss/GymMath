# GymMath
фитнес калькулятор

Веб-сайт для расчёта физических-показателей: ИМТ, BMR, TDEE, процент жира, идеальный вес, норма воды, зоны пульса, силовой максимум, темп бега, прогноз дефицита/профицита и каталог спортивных залов Саратовской области.

##  Функционал
- Профиль — пол, возраст, рост, вес, уровень активности
- 12 калькуляторов:
  - ИМТ (индекс массы тела)
  - BMR + КБЖУ (базовый метаболизм)
  - TDEE (суточная калорийность)
  - Процент жира (метод Navy)
  - Идеальный вес (формула Робинсона)
  - WHtR (индекс талия-рост)
  - Норма воды
  - Расход калорий по MET
  - Зоны пульса (Карвонен)
  - 1ПМ и рабочие веса (формула Эпли)
  - Темп бега
  - Прогноз дефицита/профицита
- Каталог залов — 55+ клубов Саратовской области

## 🛠 Технологии
- HTML5, CSS3, JavaScript
-  адаптивная вёрстка
